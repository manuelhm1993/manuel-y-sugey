# Manuel & Sugey 💙🩷

> *"Hay amores que no empiezan el día que se nombran.  
> Empiezan el día que alguien decide que la otra persona vale la pena del caos."*

Sitio web narrativo que cuenta la historia de amor de Manuel y Sugey —  
un Leo y una Sagitario, un programador y una repostera,  
completamente opuestos y completamente hechos el uno para el otro.

🌐 **[Ver en vivo](https://manuelhm1993.github.io/manuel-y-sugey/)**

## Features

- Línea de tiempo narrativa · 2011 → 2026
- Contador en vivo desde el 27 de mayo de 2025
- Puente SVG animado por scroll
- Música ambiental con overlay de bienvenida
- Sección de contrastes · Leo vs Sagitario
- Cards expandibles con perspectivas de cada uno
- Totalmente responsive · Mobile first
